1. What is VueJs ?

VueJs is a progressive framework used to building dynamic user interface and Single Page Applications. The core library is mainly focused on view layer only and it is easy to pick-up and integrate with other libraries or existing applications.

Major feature's of VueJS

i. Virtual DOM :It uses virtual DOM similar to other frameworks like ReactJs, Ember.js..Virtual DOM is a light weight in-memory tree representation of the original HTML DOM and it updated without affecting it's original DOM.
ii. Components :It is used to create a reusable custom elements in VueJs applications.
iii. Routing :Navigation between pages is acheived through Vue-router.
iv. Templates:It provides HTML based templates that binds DOM with the Vue Instance data.
v. Light Weight:VueJs is a light weight library compared to a other frameworks.

What is vue-resource?

Vue-resource is a plug-in for vuejs. This plug-in is used with vuejs, it makes web request or JSONP, in which XMLHttpRequest or JSONP. vue-resoucre has many features like Promise API or URL Templates.
