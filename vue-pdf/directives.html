v-once directive

It is used to avoid unwanted re-renders of an element. It treats the element as a static content after rendering it once for the first time. This improves performance as it does not have to be rendered again.

v-bind directive

It is used to bind one or more attributes, components prop to an element. If that attribute is bonded to our data defined in vue instance, then dynamically changes can be observed as a data changes.
