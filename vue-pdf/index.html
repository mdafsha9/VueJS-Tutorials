What is VueJS ?

VueJs is a progressive framework and it is used to build dynamic user interface and Single Page Applications. The core library is mainly focused on view-layer only. It is easily pick-up and intergrates with other libraries or existing projects.

  • VueJS let we extend HTML with HTML attributes called directives
  • VueJs directives offers functionality to HTML applications
  • VueJs provides built-in directives and user defined directives

Advantages of using VueJS

Small-in-size

The size of the framework is 18kb to 21kb and it takes no time for user download and use it.

Simple Integration

It is easily integrated with other libraries or existing projects

Flexibility

The flexibility also makes easy to understand for the developers of ReactJS, AngularJS and any other new JavaScript frameworks.

Easy to understand

One of the reason for the popularity of this framework is easy to understand. The user can easily add vuejs on his project because of the simple structure.

Virtual DOM

It uses virtual DOM similar to the other existing projects like ReactJS, AngularJS etc. Virtual DOM is a light weight in-memory tree representation of the original HTML DOM and it updates without effecting original HTML DOM.

Components

It is used to create reusable custom elements in Vuejs application

Two-way data binding

VueJS facilities Two-way communication because of "MVVM" architecture which make's it quite easy to handle HTML blocks.
