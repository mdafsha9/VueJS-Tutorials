Components

Vue Components are one of the most important feature in Vuejs that creates custom elements, which can be reused in HTML.

we have two options when defining component name:<
1.Kebab-case:
Vue.component('test-component', {
})
 2..Pascal case:
Vue.component('TestComponent', { })

Components can be reused as many times as we need

passing data from parent to child components with props

Creating a component blog-post. That's the problem components won't be useful unless you pass data to it.
Such as the title and content of specific post we want to display. That's where prop's come in.
Props are custom attributes you can register on a component. When a value is passed to prop attribute, it becomes property on that component instance
